# PDF Tools

Free online PDF utility website.

## Tech Stack
- HTML
- CSS
- Vanilla JavaScript
- pdf-lib
- PDF.js
- Tesseract.js

## Project Structure

- `/tools` — PDF utility pages
- `/css` — Stylesheets
- `/js` — Shared JavaScript
- `/js/tools-logic` — Tool-specific logic
- `/assets` — Icons, images, fonts
- `/lib` — Third-party JavaScript libraries
- `/backend` — Future Node.js backend

## Tool Categories

- Organize
- Optimize
- Convert to PDF
- Convert from PDF
- Edit
- Security
- PDF Intelligence

## Development

Frontend tools are built with HTML, CSS, and vanilla JavaScript.

Backend services will be added later for tasks requiring server-side processing.
